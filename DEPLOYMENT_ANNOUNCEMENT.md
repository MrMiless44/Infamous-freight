## ⌨️ Quick Start: Keyboard Shortcuts

Get productive immediately with these shortcuts:

**Essential:**
- `⌘K` or `Ctrl+K` - Open command palette
- `⌘/` or `Ctrl+/` - Quick search
- `?` - Show all shortcuts
- `Esc` - Close modals/dismiss

**Navigation:**
- `g h` - Go to home/dashboard
- `g d` - Go to documents
- `g s` - Go to shipments
- `g u` - Go to users
- `Alt + ←` - Navigate back
- `Alt + →` - Navigate forward

**Actions:**
- `n` - Create new item
- `/` - Focus search
- `e` - Edit current item
- `d` - Delete/archive
- `s` - Save changes
- `Ctrl + Enter` - Submit forms

**Accessibility:**
- `Tab` - Navigate forward
- `Shift + Tab` - Navigate backward
- `Space` - Activate buttons
- `Enter` - Submit/confirm
- `Escape` - Cancel/close
