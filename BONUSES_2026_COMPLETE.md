## 🔧 TECHNICAL IMPLEMENTATION

### File Structure

```
api/src/
├── data/
│   ├── bonusesSystem.js          # Main bonus configuration (3,200+ lines)
│   └── rewardsTiers.js           # Loyalty tier definitions (800+ lines)
└── services/
    ├── bonusEngine.js            # Bonus calculation engine (900+ lines)
    └── loyaltyProgram.js         # Loyalty management service (700+ lines)
```

### Core Services

#### BonusEngine (`bonusEngine.js`)
**Responsibilities**: Calculate all bonus types and totals

**Key Methods**:
- `calculateReferralBonus(referralDetails)` - Referral bonus amounts
- `calculateLoyaltyPoints(purchaseDetails)` - Points earned
- `calculateDriverPerformanceBonus(driverMetrics)` - Performance rewards
- `calculateMilestoneBonus(customerStats)` - Milestone achievements
- `calculatePromotionalBonus(customerInfo)` - Seasonal promotions
- `calculateTotalBonus(customerId, details)` - Complete package
- `determineLoyaltyTier(customerStats)` - Tier qualification
- `validateBonusRedemption(redemptionRequest)` - Redemption validation

#### LoyaltyProgram (`loyaltyProgram.js`)
**Responsibilities**: Manage tier progression and member lifecycle

**Key Methods**:
- `enrollCustomer(customerId, initialTier)` - New member setup
- `recordActivity(customerId, activity)` - Track customer activities
- `checkTierUpgrade(customerStats)` - Check upgrade eligibility
- `checkTierDowngrade(customerStats)` - Monitor tier maintenance
- `getTierBenefits(tier)` - Show tier benefits
- `calculateMonthlyBonus(currentTier)` - Monthly bonus calculation
- `redeemPoints(customerId, redemptionRequest)` - Points redemption
- `getTierUpgradeProgress(customerStats)` - Progress tracking
- `generateLoyaltyReport(customerId, customerData)` - Member reports

## 📋 USAGE EXAMPLES

### Example 1: Calculate Referral Bonus

```javascript
const bonusEngine = new BonusEngine();

const result = await bonusEngine.calculateReferralBonus({
  type: 'customer',
  referrerInfo: { loyaltyTier: 'gold' },
  referredInfo: { email: 'customer@example.com' },
  shipmentCount: 1, // Referred customer completed first shipment
});

// Result:
// {
//   success: true,
//   referralType: 'customer',
//   referrer: { bonus: 100.00, status: 'earned', tieredBonus: 100 },
//   referred: { bonus: 25.00, status: 'pending', expiryDays: 30 }
// }
```

### Example 2: Calculate Customer Loyalty Points

```javascript
const result = await bonusEngine.calculateLoyaltyPoints({
  amount: 250.00,
  customerId: 'cust_123',
  loyaltyTier: 'silver',
  isSubscriptionRenewal: true,
  promoCode: null,
});

// Result:
// {
//   success: true,
//   points: 425,           // 250 * 1.25 + 100 monthly bonus
//   value: 21.25,          // 425 points * $0.05 per point
//   tier: 'silver',
//   breakdown: {
//     basePoints: 312.5,
//     bonuses: { monthly: 100 }
//   },
//   expiryDate: Date (18 months from now)
// }
```

### Example 3: Determine Customer Tier

```javascript
const result = await bonusEngine.determineLoyaltyTier({
  points: 5500,
  monthlySpend: 300,
  totalSpend: 3200,
  shipmentCount: 120,
  accountAgeMonths: 14,
});

// Result: { tier: 'gold', qualifies: true }
```

### Example 4: Process Loyalty Signup

```javascript
const loyaltyProgram = new LoyaltyProgram();

const enrollment = await loyaltyProgram.enrollCustomer(
  'cust_456',
  'bronze'
);

// Result:
// {
//   success: true,
//   customerId: 'cust_456',
//   tier: 'bronze',
//   enrollmentDate: Date,
//   pointsBalance: 0,
//   status: 'active'
// }
```

### Example 5: Generate Loyalty Report

```javascript
const report = await loyaltyProgram.generateLoyaltyReport(
  'cust_789',
  {
    currentTier: 'silver',
    points: 2340,
    pointsBalance: 1650,
    lifetimeSpend: 1875,
    totalShipments: 45,
    membershipMonths: 8,
    recentActivityDays: 12,
  }

// Result includes:
// - Tier status with benefits

// - Monthly bonuses
```
