## 💡 Pro Tips

### Docker Compose Deployment
```bash
# View real-time logs
docker-compose logs -f api web

# Restart specific service
docker-compose restart api

# Clean up after testing
docker-compose down
docker volume prune
```

### Local Development Deployment
```bash
# Run in watch mode
pnpm dev

# Debug specific package
cd apps/api && pnpm dev --inspect

# View logs
tail -f apps/api/logs/combined.log
```

### Performance Optimization Checks
```bash
# See query count per endpoint
grep -E "SELECT|INSERT|UPDATE" apps/api/logs/combined.log | wc -l

# Average response time
grep "duration" apps/api/logs/combined.log | jq '.duration' | \
  awk '{sum+=$1; count++} END {print sum/count}'

# Error rate (last 100 requests)
tail -100 apps/api/logs/combined.log | grep '"level":"error"' | wc -l
```
