# Legal Foundation Summary

## Master Service Agreement (MSA) — Summary

- **Scope**: SaaS access + AI automation
- **Billing**: Subscription + metered usage
- **Availability**: Commercially reasonable uptime
- **Liability**: Capped (fees paid, standard)
- **Termination**: For cause or convenience
- **Governing law**: Select the operating state

## Data Processing Addendum (DPA)

- Company owns all operational data
- Infæmous Freight Enterprises is the data processor
- Sub-processors: Supabase, Stripe, cloud providers
- Data deletion/export on request

## Marketplace Addendum (Get Truck’N)

- Separate terms
- No payouts enabled until Stripe Connect phase
- Platform acts as facilitator, not carrier
