# Phase 6 / Phase 7 History Index

completion documents that live at the repository root. It exists so that
`docs/history/` — the existing home for point-in-time project snapshots —
surfaces those files in one place without requiring a coordinated bulk move.

The root-level files are retained at their current paths to keep existing links
(external trackers, past PRs, onboarding notes) working. New phase snapshots
should be added here and written directly into `docs/history/` going forward.
