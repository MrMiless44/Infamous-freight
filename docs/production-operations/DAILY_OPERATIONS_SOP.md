# Daily Operations SOP

## Morning

- Review active loads
- Confirm driver locations
- Confirm pickup appointments
- Check new quote requests
- Follow up with shipper leads
- Verify carrier paperwork

## Midday

- Track active shipments
- Update shippers
- Source carriers
- Negotiate rates
- Book new loads
- Handle delays

## End of Day

- Collect PODs
- Send invoices
- Update CRM
- Update TMS
- Log carrier performance
- Review gross margin
