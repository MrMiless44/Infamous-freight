# Operating Model

## Core Model

Infamous Freight operates as a freight brokerage and logistics coordination company.

Execution flow:

```text
Find freight -> Quote shipper -> Source carrier -> Dispatch load -> Track shipment -> Confirm delivery -> Invoice -> Repeat
```

## Revenue Model

```text
Shipper Rate - Carrier Rate = Gross Profit
```

## Load Lifecycle

| Stage | Action |
|---|---|
| Lead intake | Quote form, phone call, email, or sales outreach |
| Quote | Calculate lane, equipment, urgency, market rate, and target margin |
| Approval | Confirm rate, commodity, pickup, delivery, and terms |
| Carrier sourcing | Find qualified carrier |
| Vetting | Confirm authority, insurance, W-9, safety, and agreement |
| Dispatch | Send rate confirmation and load details |
| Tracking | Confirm pickup, transit, ETA, and delivery |
| POD | Collect proof of delivery |
| Billing | Invoice shipper and process carrier payment |
| Follow-up | Ask shipper for the next lane |

## Operating Standard

Compete on speed, discipline, communication, and execution. Do not dispatch unvetted carriers.
