## Configuration Files

Configuration files are organized in the `configs/` directory at the repository
root:

- **`configs/docker/`** - Docker Compose configurations
- **`configs/ci-cd/`** - CI/CD platform configurations (Vercel, Render, Fly.io,
  etc.)
- **`configs/linting/`** - ESLint, Prettier, and other code quality tools
