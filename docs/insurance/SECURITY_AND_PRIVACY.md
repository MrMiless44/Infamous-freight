# Security & Privacy

Insurance documents and metadata are sensitive. The Insurance Center enforces:

- **RBAC controls**: carrier users can upload and view their own records; admins
  verify and configure requirements.
- **Audit logging**: every upload, verification, rejection, and enforcement
  action is recorded.
- **Data minimization**: store only the metadata required for compliance and
  underwriting workflows.

## Recommended Controls

- Encrypt document URLs and access tokens.
- Configure retention policies for expired certificates.
- Monitor audit logs for suspicious access patterns.
