## Integration Examples

### Before: Plain HTML (❌ Not User-Friendly)
```tsx
<form>
  <input placeholder="Name" />
  <button>Save</button>
</form>
```

### After: User-Friendly Components (✅)
```tsx
<form>
  <Input
    label="Full Name"
    placeholder="Your name"
    hint="We'll use this for notifications"
    required
  />
  <Button variant="primary" type="submit">
    Save Changes
  </Button>
</form>
```

### Before: Poor Error Handling (❌)
```tsx
{error && <p style={{ color: 'red' }}>{error}</p>}
```

### After: User-Friendly Error (✅)
```tsx
<Alert
  type="error"
  title="Couldn't save"
  message="Please check your connection and try again."
  action={{
    label: "Retry",
    onClick: handleRetry
  }}
/>
```
