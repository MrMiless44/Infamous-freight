## 🔐 Authentication Quick Reference

### Generate JWT Tokens

```bash
node -e "
const jwt = require('jsonwebtoken');
const secret = process.env.JWT_SECRET || 'dev-secret';

const shipper = {
  sub: 'shipper-123',
  email: 'shipper@example.com',
  role: 'SHIPPER',
  scopes: ['shipper:create', 'shipper:checkout', 'shipper:subscribe', 'shipper:portal']
};

const driver = {
  sub: 'driver-456',
  email: 'driver@example.com',
  role: 'DRIVER',
  scopes: ['driver:location', 'driver:vehicle', 'driver:view', 'driver:accept', 'driver:deliver']
};

console.log('SHIPPER:', jwt.sign(shipper, secret, { expiresIn: '24h' }));
console.log('DRIVER:', jwt.sign(driver, secret, { expiresIn: '24h' }));
"
```

### Scope Requirements by Endpoint

```
POST   /drivers/location          → driver:location
POST   /drivers/vehicles          → driver:vehicle
GET    /jobs                      → driver:view
POST   /jobs                      → shipper:create
POST   /jobs/:id/checkout         → shipper:checkout
POST   /jobs/:id/accept           → driver:accept
POST   /jobs/:id/delivered        → driver:deliver
POST   /subscribe                 → shipper:subscribe
GET    /portal                    → shipper:portal
```

## 🏃 Quick API Tests

### Create Job

```bash
curl -X POST http://localhost:4000/api/marketplace/jobs \
  -H "Authorization: Bearer $SHIPPER_JWT" \
  -H "Content-Type: application/json" \
  -d '{
    "shipperId": "shipper-123",
    "pickupLat": 34.0522, "pickupLng": -118.2437,
    "dropoffLat": 34.0589, "dropoffLng": -118.2359,
    "estimatedMiles": 5.2, "estimatedMinutes": 25,
    "requiredVehicle": "VAN", "weightLbs": 500, "volumeCuFt": 50
  }'
```

### List Jobs (Paginated)

```bash
curl "http://localhost:4000/api/marketplace/jobs?page=1&limit=10" \
  -H "Authorization: Bearer $DRIVER_JWT"
```

### Accept Job (Atomic Transaction)

```bash
curl -X POST http://localhost:4000/api/marketplace/jobs/accept \
  -H "Authorization: Bearer $DRIVER_JWT" \
  -H "Content-Type: application/json" \
  -d '{"jobId": "job-123", "driverUserId": "driver-456"}'
```

### Checkout with Idempotency

```bash
curl -X POST http://localhost:4000/api/marketplace/jobs/$JOB_ID/checkout \
  -H "Authorization: Bearer $SHIPPER_JWT" \
  -H "Idempotency-Key: job-$JOB_ID-$(date +%s)"
```

## 📊 State Machine Transitions

```
DRAFT → REQUIRES_PAYMENT (price computed)
        ↓
    OPEN (waiting for driver)
        ↓
    ACCEPTED (driver claimed)
        ↓
    PICKED_UP (in transit)
        ↓
    DELIVERED (at destination)
        ↓
    COMPLETED (finalized)
```

**Invalid transition?** Error: `"Cannot transition from X to Y"`

## 🔄 Webhook Retry Logic

```
Attempt 1 → Fails (DB timeout)
  ↓ wait 1000ms
Attempt 2 → Fails (network error)
  ↓ wait 2000ms
Attempt 3 → Success ✓ (or final failure)
```

**All webhook events automatically retry with exponential backoff**

## 🔑 Idempotency Keys

```javascript
// Client sends
-H "Idempotency-Key: checkout-job-123-20260115"

// API logs
[webhook] idempotencyKey=checkout-job-123-20260115

// Stripe deduplicates
// Same key = same session (no duplicate charge)
```

**Use for:** POST /checkout (payment operations)

## 🐛 Debugging with Correlation IDs

```bash
# Find all logs for a request
grep "correlationId: abc-123-def-456" apps/api/combined.log

# Track webhook processing
grep "correlationId: abc-123-def-456" apps/api/combined.log | grep -E "received|processing|completed"

# Check retry attempts
grep "webhook.*retry" apps/api/combined.log | tail -20
```

**Format:** UUID v4, auto-generated by Stripe, logged in all operations

## ⚠️ Common Errors & Fixes

| Error                      | Cause            | Fix                                   |
| -------------------------- | ---------------- | ------------------------------------- |
| "Missing bearer token"     | No auth header   | Add `-H "Authorization: Bearer $JWT"` |
| "Invalid or expired token" | Bad JWT          | Regenerate with correct JWT_SECRET    |
| "Insufficient scope"       | Wrong role       | Use correct token (shipper vs driver) |
| "Price has changed"        | Stale data       | Create new job with current prices    |
| "Job not available"        | Already accepted | Try different job                     |
| "Too many requests"        | Rate limited     | Wait 15 minutes                       |
