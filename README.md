AI-powered freight logistics platform with real-time shipment tracking, carrier networks, and intelligent load orchestration.

## Overview

Infamous Freight is a modern logistics automation platform designed to connect carriers, brokers, and shippers through real-time operations software and intelligent workflow execution.

## Core Capabilities

- Real-time shipment tracking
- Carrier network management
- Intelligent load orchestration
- Automated dispatch workflows
- AI-assisted logistics operations
- Multi-tenant SaaS platform architecture

## Architecture

Infamous Freight is designed as a modular SaaS platform.

### Frontend
- Next.js web application
- React user interface

### API
- Node.js backend services
- REST and workflow endpoints

### Data
- PostgreSQL
- Prisma ORM

### Infrastructure
- Docker containers
- GitHub Actions CI/CD
- cloud deployment workflows
