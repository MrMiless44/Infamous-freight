## 🧮 Pricing Calculator

### Quick Estimate Example

**Shipment Details:**

- Weight: 50 kg
- Distance: 200 km
- Region: US-East
- Service: Express Overnight
- Insurance: Yes ($500 value)

**Calculation:**

1. Base domestic: $2.50
2. Express surcharge: $35.00
3. Weight (50 kg @ $0.40/kg): $20.00
4. Distance (200 km @ $0.05/km): $10.00
5. Insurance (2% of $500): $10.00
6. Regional multiplier (1.0x): $77.50

**Total: $77.50**

---

### Use the Pricing Engine

```javascript
// In your code
const pricingEngine = require("./services/pricingEngine");

// Calculate shipment pricing
const quote = await pricingEngine.calculateShipmentPrice({
  weight: 50,
  distance: 200,
  region: "US-East",
  serviceLevel: "express",
  freightType: "general",
  hasInsurance: true,
  insuranceValue: 500,
});

console.log(quote.totalPrice); // $77.50
```
