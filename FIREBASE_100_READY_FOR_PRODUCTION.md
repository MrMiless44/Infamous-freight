## 🚀 To Go Live in Production

### Step 1: Create Firebase Project (5 min)
Visit https://console.firebase.google.com/ and create a new project

### Step 2: Download Credentials (5 min)
- Service account JSON
- iOS GoogleService-Info.plist
- Android google-services.json

### Step 3: Configure Environment (5 min)
Update .env files with Firebase project details

### Step 4: Deploy Rules (5 min)
```bash
firebase deploy --only firestore:rules,storage:rules,firestore:indexes
```

### Step 5: Test & Launch (10 min)
```bash
./scripts/test-firebase-notifications.sh
pnpm dev
```

**Total Time to Production**: ~30 minutes
